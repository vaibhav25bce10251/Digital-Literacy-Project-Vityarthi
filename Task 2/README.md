## Task 2 – Build Your Student Digital Portfolio

For Task 2, I created and updated my student digital portfolio on three platforms: GitHub, Kaggle, and ResearchGate.

- GitHub helps me store coding projects, academic files, and technical work in an organized way.
- Kaggle is useful for learning data science concepts, exploring datasets, and improving practical problem-solving skills.
- ResearchGate is a useful academic platform where students can explore research work, publications, and scholarly discussions.

On GitHub, I created my profile and understood how repositories can be used to present projects and technical work. On Kaggle, I explored how the platform supports learning through datasets, notebooks, and competitions. On ResearchGate, I understood how academic platforms can help students connect with research-oriented content and learn more about higher studies and technical fields.

This task helped me understand that building a digital portfolio is important from the first year itself. These platforms can be improved over time and can become useful for projects, internships, skill development, and future academic and career opportunities.
